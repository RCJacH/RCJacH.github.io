今天做了个小脚本，随机生成旋律方面的一些参数，这样每天可以练习一些旋律创作。

随机生成旋律属性脚本地址

点击左侧偏上的Run按钮来运行，会在右侧显示结果。

这个脚本生成一下几个参数：

Key = 调。目前定义为五线谱调号，以后可以考虑再生成调式/音阶参数。

Number of Points = 旋律线中转折点的数量，用来决定下面的旋律形状。

旋律形状：从音阶几级上行或下行到几级。

Prominent Interval = 旋律中比较重要的音程，或者重复比较多的音程。

Number of Bars = 小节数。
